Установка venv
sudo apt install python3-venv

Создание виртуального окружения командой 
python3 -m venv my_venv

Если вы хотите создать папку с окружением в конкретной директории, 
то вместо названия окружения нужно указать путь к папке. 
Например: 
python3 -m venv ~/my_venv

Запуск виртуального окружения
source ./bin/activate

Чтобы выйти из виртуального окружения, необходимо ввести команду 
deactivate
