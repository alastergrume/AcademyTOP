мне нужно склонировать репозиторий с github, собрать образ в докер и запустить на Ubuntu

Для выполнения вашей задачи вам потребуется установить Git (если он еще не установлен), Docker и затем выполнить несколько команд. Ниже приведены шаги, которые помогут вам сделать это на системе Ubuntu.

Шаг 1: Установка Git
Откройте терминал и выполните следующую команду для установки Git:

$ sudo apt update
$ sudo apt install git

Шаг 2: Установка Docker
Установите необходимые пакеты для поддержки установки через HTTPS:

$ sudo apt install apt-transport-https ca-certificates curl software-properties-common

Добавьте официальный ключ GPG проекта Docker:

$ curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

Добавьте репозиторий Docker в список источников пакетов:

$ sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

Обновите список пакетов и установите Docker:

$ sudo apt update
$ sudo apt install docker-ce

Чтобы использовать Docker без прав суперпользователя, добавьте вашего пользователя в группу Docker. Замените username на ваше имя пользователя:

$ sudo usermod -aG docker username

Перезагрузитесь или выйдите из системы и войдите обратно, чтобы применить изменения в членстве группы.

Шаг 3: Клонирование репозитория с GitHub
Клонируйте репозиторий, используя Git. Замените URL на адрес репозитория на GitHub:

git clone URL

Перейдите в склонированный репозиторий:

cd название_репозитория

Шаг 4: Сборка образа Docker
Если в корне репозитория есть файл Dockerfile, вы можете собрать образ Docker, выполнив:

docker build -t имя_образа .

Замените имя_образа на имя, которое вы хотите дать образу.

Шаг 5: Запуск образа в контейнере Docker
После сборки образа запустите его:

docker run -d --name имя_контейнера имя_образа

Замените имя_контейнера на имя для вашего контейнера и имя_образа на имя образа, которое вы задали при сборке.

Теперь вы успешно клонировали репозиторий, собрали образ Docker и запустили его в виде контейнера на Ubuntu.
